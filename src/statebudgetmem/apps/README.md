# Applications

This directory is reserved for the **final StateBudgetMem system demo**.

The existing baseline-only Gradio comparison is intentionally located at:

```text
statebudgetmem/baselines/memorybank/demo.py
```

Once the end-to-end pipeline is ready, add the project-level visualization here,
for example `statebudgetmem_demo.py`, showing the version timeline, current and
historical views, routing decision, selected memories, token budget, and stale
memory warnings.
