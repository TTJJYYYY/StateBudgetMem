# Contributing

This is a four-person research project. Read these files before editing code:

1. `docs/StateBudgetMem_research_plan.md`
2. `docs/ARCHITECTURE.md`
3. `docs/TEAM_WORKFLOW.md`

Use the shared contracts from `statebudgetmem.interfaces`, place tests in the
mirrored `tests/` folder, run `pytest -q`, and keep each commit focused on one
module or task.
